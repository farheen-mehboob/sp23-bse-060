Semester 1 Project: C++ Library Management System

Project Name

Library Management System in C++

Summary

This project is a console-based library management system developed using C++ programming language. It allows the user to manage the library's inventory, issue and return books, and maintain a database of issued books.

Description

The system features include adding new books, deleting existing books, updating the book inventory, issuing books to library members, and generating reports. The interface is entirely console-based, allowing interaction through terminal inputs.

Semester 2 Project: Online Shopping Website

Project Name

Online Shopping Website using Java OOP

Summary

This project is a fully functional e-commerce website developed using Java and Object-Oriented Programming principles. It includes product browsing, shopping cart management, and order processing features.

Description

The website allows users to browse through different product categories, add items to their cart, and proceed to checkout. It uses Java OOP concepts like inheritance, polymorphism, and encapsulation for a modular and scalable design.
